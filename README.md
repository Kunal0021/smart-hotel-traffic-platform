
# Smart Hotel & Traffic Intelligence Platform


## Features

- JWT Authentication
- Role Based Access Control
- Hotel Management
- Booking Engine
- Admin Analytics Dashboard
- Revenue Visualization
- AI Hotel Recommendation Engine
- Traffic Aware Ranking Algorithm
- Real Time Updates using Socket.IO
- Swagger API Documentation
- Dockerized Deployment
- CI/CD using GitHub Actions


## Tech Stack

Frontend:
React
TailwindCSS
Socket.IO Client
Recharts

Backend:
Node.js
Express
MongoDB
JWT
Socket.IO
Swagger


## AI Recommendation Engine

Uses weighted scoring based on:

- Location preference
- User budget
- Hotel rating
- Room availability
- Traffic congestion
- Booking popularity
